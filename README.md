Application E-Commerce en Java
Description

Cette application est une plateforme de commerce électronique développée en Java dans le cadre d'un projet académique. Elle permet aux utilisateurs de consulter des produits, gérer leur panier et effectuer des commandes à travers une interface conviviale.

Fonctionnalités
Authentification des utilisateurs
Gestion des produits
Recherche et filtrage des articles
Gestion du panier d'achat
Validation des commandes
